# capitalPythonAPI
This is a middle ware use to wrap the dll API provided by cpital.com.tw

# Requirements
OS requirement:
Windows

Python requirement:
ctypes<br/>
pandas<br/>
clr<br/>


# Usage: 
Provide key.config with the following content
>username=yourUserName<br/> 
>password=yourPassword
## Execution
./runServer.bat
